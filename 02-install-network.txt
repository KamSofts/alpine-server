NETWORK SETTING:-
~~~~~~~~~~~~~~~~~
    ~# vi /etc/network/interface
        // add this lines in interface file
        auto eth1
        iface eth1 inet dhcp
        esc :wq enter // to save and exit from interface file
    ~# poweroff
